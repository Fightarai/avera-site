What is Avera?

Avera is a next-generation AI-powered trading engine designed to execute, adapt, and evolve. It blends proven quantitative strategies with large language model (LLM) intelligence to dynamically interpret data, sentiment, and macroeconomic conditions — all in real time.

Avera is not just a bot — it’s an autonomous system that thinks, learns, and trades like a pro.

Core Features

  • Adaptive quant strategies: RSI, VWAP, Mean Reversion
  • LLM-enhanced signal logic (GPT/FinGPT integrations)
  • Risk-aware execution with auto scaling
  • Modular backtesting and forward-testing engine
  • Strategy generation and review via LLM prompt injection
  • Built-in architecture for dashboard and governance layers

System Architecture

Avera’s modular architecture features a plug-and-play strategy core (Freqtrade), an optional LLM assistant layer (OpenAI, FinGPT, DeepSeek), a rebalancing engine, and integration points for dashboards, wallets, and governance modules.

Sample Strategies

  • RSI + LLM Bias Filter: Filter RSI oversold triggers using LLM macro sentiment
  • VWAP Pullback + Trend Confirmation: Trade volume-weighted reversals with trend guards
  • LLM Position Sizing: Adjust lot sizes based on predicted volatility and sentiment shift
  • FinGPT Signal Composer: Combine economic data with price action

LLM Integration

Avera integrates with large language models (LLMs) to enhance trade logic beyond indicators. It can ingest sentiment data, macroeconomic narratives, social signals, and even strategy templates — converting them into actionable triggers. We support integrations with GPT-4, FinGPT, and custom local LLMs.
