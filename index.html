What is Avera?

Avera is an AI-powered engine for automated trading, fusing quant strategies with LLM-generated insights and smart execution logic.

Core Features

  • Quant strategies: RSI, VWAP, Mean Reversion
  • LLM sentiment overlay and adaptive logic
  • Risk-adjusted position sizing
  • Modular backtesting and live deployment

Sample Strategies

  • RSI Oversold Entry + LLM Bias Overlay
  • VWAP Pullback with Trend Confirmation
  • LLM-Suggested Risk Multiplier

LLM Integration

LLMs (like GPT or FinGPT) help guide trade entry/exit, adjust signals based on macro, sentiment, and even suggest new strategies through code generation.
